Detailed Description

Represents a grid in the radial distance to the nucleus.

Can be a logarithmic Grid or a linear Grid.

tWhether the Grid is linear, exponential or linear + exponential. If this is set to expGrid, the Grid is r = exp(rmin + dx*i). If it is set to linGrid, it is r = rmin + dx*i for i = 0..N-1. If it is set to linExpGrid, the grid is such that rmin + i*dx = rmin*r + alpha*log(r)
