virtual void SCF::calculateFMatrix ( std::vector< MatrixXld > &  F,
std::vector< MatrixXld > &  K,
std::vector< MatrixXld > &  C,
std::vector< ldouble > &  E 
)
protectedpure virtual

Calculate F matrix, which represents the Hamiltonian using the Numerov method.

K is the inverse of F. This is used for the Gordon and renormalised methods, since these matrices are calculated per Grid point. The sparse method uses a large matrix solving all points simultaneously.

Parameters
FTo be returned by reference. Matrix F for each Grid point.
KTo be returned by reference. Inverse of F.
CTo be returned by reference. Independent term.
EValues of energy in each orbital.

Implemented in DFT, HF, and HFS.
